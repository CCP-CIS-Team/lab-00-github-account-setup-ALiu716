# How to Submit This Lab
### First time using GitHub? You are in the right place.

---

## The 4-Command Workflow (Every Lab This Semester)

Once you have completed your work, submitting is always these same 4 commands:

    git add .
    git commit -m "lab00 complete - Your Full Name"
    git push

That is it. Every lab. Same 4 steps.

---

## One-Time Setup (Do This Once — Right Now)

Install Git: https://git-scm.com/download/win (Windows) or run git --version on Mac

Tell Git who you are:

    git config --global user.name "Your Full Name"
    git config --global user.email "your-email@myCCP.edu"

---

## Confirm It Worked

Go to your repo on github.com — if your answers are visible in the README, Professor Chang can see them automatically. You are done.

## Troubleshooting Quick Reference

| Problem | Fix |
|---|---|
| git command not found | Install Git first (see README Step 3) |
| Permission denied | Use HTTPS URL not SSH |
| Push asks for password | Use a Personal Access Token, not your GitHub password |
| Cannot find repo URL | Check email for Classroom invite |
